# Retrieval Tuning 평가 기준선 (Before)

## 평가 기준

### 판정 등급
- **양호**: 상위 5건 중 3건+ 질의 의도 직접 일치. 노이즈 1건 이하.
- **대체로 양호**: 상위 5건 중 2건+ 일치. 경미한 혼입 있으나 핵심은 맞음.
- **실패**: 상위 5건 중 질의 의도 직접 일치 1건 이하. 또는 완전 무관 사건 지배.

### 평가 축 (4개)
1. **질문 의도 적합성**: top-5가 질의 핵심 쟁점에 맞는가
2. **복합 조건 회수력**: 2개+ 축 조합 질의에서 양쪽 다 잡는가
3. **noise 감소**: exclusion으로 무관 사건이 걸러지는가
4. **경계형 질의 설명력**: 미묘한 구분(양정 vs 비위, 성립 vs 보복)을 잡는가

## 재평가 체크리스트 (tuning 후)

tuning 후 동일 13질의를 재실행하고 아래를 확인:

- [ ] Q7: 수습 사건 혼입 줄었는가
- [ ] Q8: 각하/구제이익 노이즈 사라졌는가
- [ ] Q2: procedure 사건이 상위에 올라오는가
- [ ] Q4: 여전히 모집단 부족인가 (retrieval로 해결 불가 확인)
- [ ] Q1,Q3,Q5,Q6,Q10,Q11: regression 없는가
- [ ] disciplinary_severity 단독 질의에서 procedural noise 줄었는가
- [ ] employment_stage regular/probation 혼입률 변화
